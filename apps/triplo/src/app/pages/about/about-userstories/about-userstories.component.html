The Use Cases for the Application!

  1. Een gebruiker krijgt projecten aanbevolen op basis van de open-source projecten die de gebruiker heeft geliket, zodat hij nog meer projecten binnen zijn interesse kan ontdekken.
  2. Een gebruiker kan een applicatie schrijven om mee te werken aan een project, zodat hij kan bijdragen aan projecten met zijn interesse.
  3. Een gebruiker kan een reactie achterlaten op een project zodat de gebruiker zijn mening kan achterlaten.
  4. Een gebruiker kan reageren op een reactie op een project, zodat hij zijn mening over de reactie kan achterlaten.
  5. Een gebruiker van een project kan een taak die hoort bij het project aan te maken zodat het duidelijk is waar ze aan het werken zijn.
  6. Een gebruiker van een project wil een sub taak aankunnen maken bij een bestaande taak zodat hij de taak verder kan opsplitsen.
